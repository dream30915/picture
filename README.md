# Picture Repository (รูป)

A repository for storing and organizing pictures/images.

## Description

This repository serves as a collection for pictures and images. It provides a structured way to organize and manage visual content.

## Structure

```
images/
├── nature/       # Nature and landscape photos
├── architecture/ # Buildings and architectural photos
├── portraits/    # Portrait photography
└── miscellaneous/ # Other images
```

- Supports various image formats (JPG, PNG, GIF, SVG, etc.)

## Usage

1. Clone the repository
2. Add your images to the appropriate directories
3. Commit and push your changes

## Contributing

Feel free to contribute by adding images or improving the organization structure.
